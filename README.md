AI App Built with Google AI Studio

This project is an AI-powered application created using Google AI Studio — a browser-based platform for building and prototyping apps using Google’s Gemini models.

📌 Overview

This app demonstrates how to leverage modern generative AI to build intelligent, interactive applications with minimal setup. It uses Google’s Gemini API to process user inputs and generate smart outputs in real time.

Google AI Studio allows developers to:

Design prompts
Build full-stack apps using natural language
Export working code (React/TypeScript)
Test AI behavior before deployment
✨ Features
🤖 AI-powered responses using Gemini models
⚡ Real-time interaction
🧠 Custom prompt engineering
🛠️ Built using no/low-code AI tooling
🌐 Easily deployable and shareable
🧰 Tech Stack
Frontend: React (auto-generated via AI Studio)
Backend: Gemini API
Platform: Google AI Studio
Language: JavaScript / TypeScript

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
